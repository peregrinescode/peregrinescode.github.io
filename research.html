A little background

A domain is simply the name of a website. It's the bit in the middle of a website address, usually after a www. and before the specific page reference. For example, this website page has an address, or universal resource locator (URL), of:

https://www.rosswarren.net/index
where rosswarren.net is the domain name and index is the page name. The www is a label which creates a subdomain of rosswarren.net, which is itself a subdomain of net. The subdomain www stands for World Wide Web and indicates that the host server it represents hopefully functions as a website.

Vacuum deposition

A domain name alone does not constitute a website. You also need some content, for example, some blog posts about what a domain is. And you need somewhere to store that content. The computer where you store it is called a host.
